# 共創カフェ運営マニュアル Webアプリケーション

デロンギ マグニフィカS（ECAM22112）を使用したカフェ向けの運営マニュアルWebアプリケーションです。

## 概要

このアプリケーションは、カフェスタッフが日々の業務を効率的に行えるよう、以下の機能を提供します：

- **4つのメニューマニュアル**：コーヒー、エスプレッソ、ホットミルク、キャラメルマキアート（各￥300）
- **清掃チェックリスト**：日次・週次・月次の清掃タスク管理
- **画像付きマニュアル**：清掃箇所の写真を表示
- **トラブルシューティング**：よくある問題と解決方法
- **オフライン対応**：ブラウザのLocalStorageを使用

## ファイル構成

```
研修マニュアルプロトタイプ/
├── index.html          # メインHTMLファイル
├── styles.css          # スタイルシート
├── script.js           # JavaScriptファイル
├── data.js             # 初期データ定義
├── README.md           # このファイル
└── images/             # 画像フォルダ
    └── README.md       # 画像の配置方法
```

## セットアップ

### 1. ファイルの配置

すべてのファイルが同じディレクトリにあることを確認してください。

### 2. ブラウザで開く

`index.html` をブラウザで開きます。

**推奨ブラウザ:**
- Google Chrome（最新版）
- Mozilla Firefox（最新版）
- Microsoft Edge（最新版）
- Safari（最新版）

### 3. 画像の配置（オプション）

提供された4つの部屋の写真を清掃マニュアルに表示する場合：

1. 提供された4枚の部屋の写真を`images/`フォルダに保存
2. ファイル名を`room1.jpg`, `room2.jpg`, `room3.jpg`, `room4.jpg`に変更
3. ブラウザでindex.htmlを開く
4. 「清掃マニュアル」→「週次清掃」タブで画像を確認

詳細は `images/README.md` を参照してください。

## 使い方

### ダッシュボード

アプリケーションのホーム画面です。

- **メニューカード**：4つのメニューに素早くアクセス
- **今日の清掃チェックリスト**：本日の清掃タスクの進捗を確認
- **クイックリンク**：よく使う機能にすぐアクセス

### メニューマニュアル

各メニュー（コーヒー、エスプレッソ、ホットミルク、キャラメルマキアート）の詳細な作り方を表示します。

**含まれる情報:**
- 価格と容量
- 所要時間
- 必要な材料
- 必要な器具
- ステップバイステップの手順
- ポイントと注意事項
- よくある問題と対処法

**機能:**
- 印刷ボタンで紙に出力可能
- 各ステップに画像を追加可能（編集モード）

### 清掃マニュアル

日次・週次・月次の清掃タスクを管理します。

**日次清掃（営業後に毎日）:**
- マシン外装の拭き掃除
- トレイ・カス受けの清掃
- 水タンクの清掃
- 抽出口の清掃
- スチーム管の清掃

**週次清掃（週に1回）:**
- 抽出ユニットの清掃
- 豆ホッパーの清掃
- 店内床・テーブルの清掃（写真付き）

**月次清掃（月に1回）:**
- 石灰除去
- 深部清掃
- 備品チェックと補充

**機能:**
- チェックボックスで完了を記録
- 完了時刻を自動記録
- 日次タスクは毎日自動でリセット
- 完了したタスクはグレー表示

### トラブルシューティング

マシンの問題と解決方法を確認できます。

**含まれる項目:**
- マシンが起動しない
- 抽出されない
- スチームが出ない
- エラーメッセージが表示される
- コーヒーの味が悪い
- 異音がする

### 設定

アプリケーションの設定とデータ管理を行います。

**データ管理:**
- **バックアップ**：すべてのデータをJSONファイルとしてダウンロード
- **復元**：バックアップファイルからデータを復元

**マニュアル編集:**
- **編集モード**：マニュアルの内容を編集可能にする（将来の機能）

**更新履歴:**
- データの変更履歴を表示

## 技術仕様

### 使用技術

- **HTML5**：構造とマークアップ
- **CSS3**：スタイルとレイアウト（Flexbox/Grid使用）
- **Vanilla JavaScript（ES6+）**：すべての機能を実装
- **LocalStorage**：メニューデータと清掃データの永続化
- **IndexedDB**：画像ファイルの保存

### ブラウザ互換性

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### データストレージ

- **LocalStorage**：
  - `cafeManual_menuData`：メニューデータ
  - `cafeManual_cleaningData`：清掃データ
  - `cafeManual_settings`：設定
  - `cafeManual_updateHistory`：更新履歴

### レスポンシブデザイン

- デスクトップ（1200px+）
- タブレット（768px-1199px）
- モバイル（767px以下）

### 印刷対応

各ページに「印刷」ボタンがあり、印刷に最適化されたレイアウトで出力できます。

## カスタマイズ

### カラースキームの変更

`styles.css` の `:root` セクションで変数を変更できます：

```css
:root {
    --primary-color: #6F4E37;      /* コーヒーブラウン */
    --secondary-color: #D4A574;    /* キャラメル */
    --accent-color: #FFFFFF;       /* ホワイト */
    --background-color: #F5F5F5;   /* ライトグレー */
}
```

### メニューの追加・編集

`data.js` の `initialData.menu` セクションを編集することで、メニューを追加・変更できます。

### 清掃タスクの追加・編集

`data.js` の `initialData.cleaning` セクションを編集することで、清掃タスクを追加・変更できます。

## トラブルシューティング

### データが保存されない

- ブラウザのプライベートモードを使用していないか確認
- ブラウザのストレージ容量を確認
- 別のブラウザで試してみる

### 日次チェックリストがリセットされない

- ブラウザの日付と時刻が正しいか確認
- ページを再読み込みしてみる

## データのバックアップ

重要なデータは定期的にバックアップすることを推奨します。

1. 「設定」ページを開く
2. 「データをバックアップ」ボタンをクリック
3. JSONファイルがダウンロードされます
4. 安全な場所に保存してください

## データの復元

1. 「設定」ページを開く
2. 「データを復元」ボタンをクリック
3. バックアップファイル（JSON）を選択
4. ページが自動的に再読み込みされます
