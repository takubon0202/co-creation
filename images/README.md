# 画像フォルダ

このフォルダには、カフェマニュアルで使用する画像を配置します。

## 清掃マニュアル用の画像

提供された部屋の写真を以下のファイル名でこのフォルダに保存してください：

1. **room1.jpg** - テーブルと椅子のクローズアップ
2. **room2.jpg** - テーブルと椅子のクローズアップ（別角度）
3. **room3.jpg** - カフェ全体の俯瞰図
4. **room4.jpg** - テーブルと椅子のクローズアップ（さらに別角度）

これらの画像は、週次清掃の「店内床・テーブルの清掃」タスクに表示されます。

## 画像の配置方法

1. 提供された4枚の部屋の写真を、このフォルダ（images/）に保存
2. ファイル名を上記の通りに変更（room1.jpg, room2.jpg, room3.jpg, room4.jpg）
3. ブラウザでindex.htmlを開くと、清掃マニュアルに画像が表示されます

## サポートされる画像形式

- JPG / JPEG
- PNG
- WebP
